#Migraine Check App
A lightweight web-based migraine self-assessment tool built with pure HTML, CSS, and JavaScript.
The app helps users quickly evaluate migraine-related symptoms using a simple questionnaire and visual score system.
Features:
-Responsive mobile-friendly design
Migraine symptom questionnaire
-Real-time score calculation
-Color-based risk visualization
-Progress bar feedback
-Dark mode support
-No backend required
-Fully compatible with GitHub Pages
-Lightweight and fast
-Privacy-friendly (no data collection)
